Getting Started

  1. Make sure the Local Chrome Server is running: local-chrome-server serve
  2. The extension will automatically connect to the server
  3. Use the CLI or REST API to control the browser: chrome-cli --help

Available Commands

  • Mouse Control: Move, click, scroll
  • Keyboard Input: Type text, press keys
  • Screenshots: Capture with mouse cursor
  • Tab Management: Open, close, switch tabs

API Endpoints

Server runs at: http://127.0.0.1:8765

  • POST /command - Execute any command
  • POST /mouse/move - Move mouse
  • POST /keyboard/type - Type text
  • POST /screenshot - Capture screenshot
  • GET /tabs - List all tabs

Troubleshooting

  • Check if server is running: chrome-cli status
  • Verify extension is enabled in Chrome extensions page
  • Check browser console for errors (F12 ‚Üí Console)
  • Restart server and refresh extension if needed
